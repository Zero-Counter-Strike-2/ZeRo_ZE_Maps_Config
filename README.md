```

- ze_boatescape888
## DataTime 
- 2024年09月09日 15:24

- Arctic_Fox
## Logs
- 1.**添加** 地图翻译
- 2.**删除** 错误的翻译
- 3.**修改** 翻译文本的语法
```

## 注意事项

- 进行 `PR` 之前请先阅读 <a rel="readme" href="地图翻译提交规范.md">地图翻译提交规范</a>
- 禁止 **单个** `PR` 修改 **多个** `地图翻译文件`
- 文本换行符 `LF`
- 文本保存格式 `UTF8-无BOM`
- 文本缩进 `4空格缩进`
- JSON/JSONC `2空格缩进`
- 文件内容**禁止**使用中文标点
